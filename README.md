# Sistema_Gerenciamento_Bancario

## Requisitos
- Java 21 (LTS)
- Maven 3.9+ (ou instalado via gerenciador de pacotes)

## Como compilar e executar
1. Construir o projeto com Maven:

```bash
cd Sistema_Bancario
mvn clean package
```

2. Executar o JAR:

```bash
java -jar target/sistema-bancario-1.0.0.jar
```

Observação: O projeto foi atualizado para usar Java 21 — ver `pom.xml`.
